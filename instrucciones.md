# Instrucciones para guardar el dataset

- Una vez clonado el repositorio crea el entorno virtual con el siguiente comando:`py -m venv env`
- Activate tu entorno virtual con el comando `env\Script\activate`
- Instalamos dependencias con el comando `pip install -r requirements.txt`
- Ejecuta el script para capturar las señas, con el comando `py tools/capture_dataset.py`
- En la pantalla se abrira la captura de la camara, para hacer la captura, hacen la seña y presionan la tecla que sale en la pantalla para hacer la captura, despues de eso les imprimira que se guardo.
- Para salir solo basta con presionar q
